## 📦 Instalasi

Clone repository:

```bash
git clone https://github.com/FikXzModzDeveloper/tools-clii.git
cd tools-clii
```

Install dependencies:

```bash
npm install
```
## jika eror gunakan 
```bash
npm run setup
```

---

▶️ Menjalankan

Jalankan script:

```bash
npm start
```
